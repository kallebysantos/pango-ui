## Installation

<Code Language="fish" class="py-2">dotnet pango add data-table</Code>

## Usage

DataTable is backed by [QuickGrid](https://learn.microsoft.com/en-us/aspnet/core/blazor/components/quickgrid), so it follows the same API reference.

But its also have some builtin `SSR` features like Pagination,
it means that you don't need to stick Interactive `@rendermode` for it.
